# Movie Info App

A modern movie discovery app built with React, TypeScript, and Tailwind CSS. Search movies, explore trending and popular titles, view detailed info and cast, and save your favorites.

## Features

- Trending & popular movies on the home page
- Search movies by title
- Movie details with overview, cast, and production info
- Favorites system with local storage persistence
- Smooth animations with Framer Motion
- Responsive design

## Getting Started

1. Clone the repo
2. Install dependencies:
   ```bash
   npm install
   ```
3. Create a `.env` file based on `.env.example` and add your TMDB API key:
   ```
   VITE_TMDB_API_KEY=your_key_here
   ```
   Get a free API key at [themoviedb.org](https://www.themoviedb.org/settings/api)
4. Run the dev server:
   ```bash
   npm run dev
   ```

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- React Router
- Zustand (state management)
- Lucide React (icons)
